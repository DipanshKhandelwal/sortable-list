<h1 align="center">Sortable List</h1>

<p align="center">Scratch implementation of a sortable list synced with firebase database !!</p>

## Clone
``` bash
git clone https://github.com/DipanshKhandelwal/sortable-list.git
cd sortable-list
```

## Run
``` bash
npm install
npm start
```
